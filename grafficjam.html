GRAFFIC JAM

What is Graffic Jam?

Graffic-Jam is a 2D beat 'em up designed for 2 players, drawing inspiration from graffiti culture. The game features multiple levels, each presenting a unique scenario related to the world of graffiti, such as train tracks or city streets.

Players aim to reach the end of each level to create a graffiti piece, encountering various waves of enemies along the way. These adversaries include police, security forces, or rival gangs that players must defeat.

Development

It is a multiplayer browser game developed with Phaser 3, designed for PC. Graffic Jam is developed with java by passing messages from an authoritative server to the clients.

Vivo and Tuerto must move and defend themselves from the enemies by attacking them using spray paint and paint brushes. When entering a new level, a 3-second countdown will appear before hordes of enemies appear.
